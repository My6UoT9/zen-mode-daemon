# zen-mode-daemon
Experimental browser to show all the web in "reading view"

---

== Maven

  $ mvn compile

  $ mvn dependency:resolve

  $ mvn exec:java


Package:

  $ mvn package

  $ java -cp target/my-app-1.0-SNAPSHOT.jar com.mycompany.app.App
